EUGM

Ontology for EUGM

Author(s): Sőrés László, SZTFH (SARA)
Version: v1.0,2025.04.25

This ontolgy contains Classes and Properties used in the SensorThings API implementation of the EUGM database. It describes mapping between the original GSEU WP4 datamodel and the FROST dissemination database. EUGM classes are specialized from SensorThings API classes. Monitoring Point, Borehole and Aquifer are subclasses of STA Thing. Different Timeseries types (measured, processed, forecasted) are sublasses of STA Datastream. Timeseries characteristics are encoded as Datastream json properties.
